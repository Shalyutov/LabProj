# Лабораторная информационная система

Проект ЛИС представляет собой библиотеку сущностей в предметной области медицинской лаборатории и набор инструментов для реализации прецедентов.
Предназначается для использования в частной клиники.

Домен определяет области лабораторных процессов:
+ заказ исследований в лаборатории (преаналитический этап)
+ работа с материалами исследований (аналитичский этап)
+ получение результатов исследований
+ работа со справочниками
+ экспорт данных во внешние системы (интеграция между разными ЛИС или МИС)
+ организационный учёт (учет организации, лабораторий и оборудования)
+ складской учёт (учет пробирок и других расходных материалов)
+ бухгалтерский учет (расчет стоимоти услуг и финансовые отчеты)
+ управление доступом (IAM)

Проект в ситуативной разработке.
Обновление от 25 ноября 2025 года.
